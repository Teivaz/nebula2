nMap Package
============

Yet another terrain implementation for Nebula2.  Uses Willem H. de Boer's
geometric mipmapping technique with a static vertex buffer to take advantage
of modern graphics cards.

Author: Ling Lo
Version: 1.03
License: Same as Nebula, see nebula/doc/source/nmap_license.txt

Building
==================
Run nebula2/updsrc.tcl
Build the project or solution thus created -- this should create nebula2/bin/[os]/nmapviewer.exe

Running the example
==================
Copy nmapexample.bat to the place where nmapviewer.exe
Run nmapexample.bat

Controls:
   see nViewer
