
** Installation **

Copy 'scripts' direcoty and all its script files under that directory 
to the '$3dsmax' installed directory.

The followings are direcotory layout after the copy.

  - 3dsmax
    - scripts
      - nebula2
        - various max script files for nmaxtoolbox package.
      - startup
        - nmaxtoolbox.ms

Whenevenr 3dsmax starts, it automatically evaluates nmaxtoolbox.ms.
