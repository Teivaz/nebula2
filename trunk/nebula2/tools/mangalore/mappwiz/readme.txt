== Mangalore Application Wizard ==

[Introduction]
  The Mangalore Application Wizard helps you to write your own application 
  which uses the mangalore::application module. See application/app.h.

[Installation]
  The only thing you need to get the GUI running is wxPython 2.5.x (or later), 
  available at www.wxpython.org.
