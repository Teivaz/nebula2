
This directory contains max script files for Nebula2 3DS Max toolkit.

The following files should be located under '$3dsmax/scripts/nebula2/'

  .export.ms           - 'Export' menu item
  .batchexport.ms      - 'Batch Expot' menu item
  .materialeditor.ms   - 'Material Editor' menu item
  .materialhandlers.ms
  .lod_ca.ms           - 'LOD' menu item
  .lod_fn.ms           - 'LOD' menu item
  .attach_ca.ms        - 'Attachment Node' menu item
  .attach_fn.ms        - 'Attachment Node' menu item
  .meshoptions_ca.ms   - 'Mesh Options' menu item
  .meshoptions_fn.ms   - 'Mesh Options' menu item
  .help.ms             - 'Help' menu item
  .about.ms            - 'About' menu item
  .custattrib.ms       - functions for add/remove custom attributes.
  .lib.ms              - various library utility functions.
  .terrain.ms          - 'Terrain Editor' menu item
  .terraineditor.ms    - 'Terrain Editor' menu item
  .n2logo256.bmp       - nebula2 logo bitmap file in about dialog.
  .character.ms        - 'Animation Editor' menu item
  .moflow.ms           - library functions for motion flow
