== Nebula2 Application Wizard ==

[Introduction]
  The Nebula2 Application Wizard helps you to write your own application 
  which uses Nebula2 application module.

[Installation]
  The only thing you need to get the GUI running is wxPython 2.5.x (or later), 
  available at www.wxpython.org.
